【01】博客搭建

安装必备软件,以下为下载地址

  • Node.js (https://nodejs.org/en/)
  • git (https://git-scm.com/downloads)

接下来把完整步骤记录一下

步骤二：安装hexo

  • hexo是基于node.js的静态博客，也是搭建在GitHub上

  • 在电脑上新建一个blog文件夹,该文件夹用于存放你的博客文件，桌面空白处右击，点击如图所示。出现命令框。

  • 打开命令框后，输入如下命令进行安装git；完成后输入hexo init

  • 紧接着启动服务(hexo init初始化hexo –> cnpm install安装生成器–>hexo clean –>hexo generate –>hexo deploy)

  • 最后打开浏览器，输入localhost:4000,就可以在本地看到你的个人博客了

【02】博客绑定个人域名

  1. 首先购买个人域名（阿里云、腾讯云、花生壳等都行）

【03】博客增加评论

我选择了valine评论系统，接下来讲述如何增加评论

第一步：注册LeanCloud

  • 进入 LeanCloud 官网，需要注册账号，然后开始创建应用，这里选择开发版 点击创建

  • 点击应用进入设置，在设置页首先点击存储，查看是否有Comment和Counter,没有则创建，权限设为无限制

  • 然后点击设置 > 安全中心 ,将除了数据存储的服务全部关闭。

  • 最后点击应用 Key 取得我们 AppKey 和 App id
